## How to use this app?

---

- clone the repository.

```
git clone https://github.com/Mr-vabs/todo-node.git
```

- change the directory.

```
cd todo-node
```

- install the node modules.

```
npm install
```

- Run using node or nodemon

`install nodemon by 'npm i -g nodemon' if you don't have.`

```
nodemon app.js 
```

  **Or**

```
node app.js
```
